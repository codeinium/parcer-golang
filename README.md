# Парсер сервиса доставки "Самокат"

Что умеет: 
  * Имитировать пользователя: Заходит на сайт, сам вводит адрес и выбирает его.
  * Работать в двух режимах: В "видимом" (headless: false для отладки) и "фоновом" (headless: true для скорости).
  * Использовать прокси: Может отправлять запросы через платные прокси, чтобы избежать блокировок.
  * Собирать данные: Проходит по указанным в конфиге категориям и собирает название, цену и ссылку на каждый товар.
  * Сохранять результат: Аккуратно складывает все найденные товары в CSV-файл, готовый для Excel.

Как запускать: 
  * Склонируйте
  * Настройте config.yaml - то есть, адрес доставки, целевые категории для парсинга, headless - с окном браузера или без (false и true соответвенно), proxy по шаблонну введите СВОИ данные для подключения к прокси
  * go mod tidy
  * go run ./cmd
  * Результат в samokat_products_go.csv
